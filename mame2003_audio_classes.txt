mame2003 per-game audio class list
======================================================================

WHAT THE CLASSES MEAN
----------------------------------------------------------------------
Class 1 (no resampling): every sound source in the game synthesizes audio
  directly at the core's chosen output rate. The mixer is a 1:1 copy; there
  is no rate conversion inside the core. (AY8910, SN76496, YM2203, YM2612,
  OKIM6295, DAC, MSM5205/ADPCM, qsound, SEGAPCM, K-series PCM, POKEY, etc.)

Class 2 (resampling needed): the sound source runs at a fixed clock-derived
  rate that is NOT the output rate, so the mixer must resample it. (YM2151,
  YM3812/YM3526/Y8950, YMF262, YM2413, YM2608, YM2610/B, NAMCO wavetable,
  C140, VLM5030, SP0250, SCSP, GAELCO.)

Per game we bucket as:
  NONE   - 0 class-2 sources              -> already no core resampling
  SINGLE - exactly 1 class-2 native rate  -> output AT that rate, resample-free
  MULTI  - >=2 distinct class-2 rates     -> some resample unavoidable
  SPECIAL- SAMPLES/CUSTOM only            -> rate is runtime/driver dependent

Rate policy: SINGLE games drive the output rate from the chip's own native
rate (read at runtime). All other buckets follow the user 'hint'.
